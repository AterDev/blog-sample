# 前端页面实现

本文档描述前端页面的实现细节。

前端项目在`src/ClientApp/WebApp`目录下。

## 技术栈

使用Angular 21实现，使用signal进行状态管理。

## 页面

页面在前端目录`/src/app/pages`下，包含`user`和`blog`两个目录，分别对应相关的页面组件。

### User页面

- 用户详情页面：查看用户详细信息
- 修改密码页面：修改用户密码

### Blog页面

- 博客列表页面：查看所有博客文章，边栏显示分类，点击可进行筛选，点击文章标题可查看详情
- 博客详情页面：查看博客文章内容，所属分类信息
- 新增博客页面：创建新的博客文章，填写标题、内容、分类等信息
- 修改博客页面：同新增页面，但会预填充已有数据
- 分类管理页面：查看、添加、修改、删除博客分类
