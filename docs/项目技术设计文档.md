# 项目内容

## 核心实体

- 用户
  - 用户名/密码/昵称/头像
  - 用户名是唯一的
- 博客
  - 标题/内容/作者
  - 标题是唯一的
- 博客分类
  - 分类名称/描述
  - 分类名称是唯一的

博客与分类是多对多的关系。

## 功能设计

- 博客分类管理：增删查改
- 用户登录/密码修改
- 添加博客，关联用户与分类
- 修改博客，标题/内容/分类
- 查看博客，列表(分类筛选)/详情

## 具体逻辑

**对于博客分类的处理**:
- 创建博客时，如果未选择分类，则默认归类到“未分类”中。
- 删除博客分类时，不会删除该分类下的博客，而是将其重新归类到“未分类”中。
